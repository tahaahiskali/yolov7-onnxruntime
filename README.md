# yolov7-onnxruntime
YOLOv7 ONNX Runtime C++ inference code

**To run code:**
```
make
./yolov7-onnxruntime
```


To make the program more flexible, args etc. You can pull request for features.


**Environments:**
```
Ubuntu 20.04.6 LTS  
CUDA 12.1  
ONNXRuntime 1.12  
OpenCV 4.2.0 ( Any version with dnn module ) 
```
