# yolov7-onnxruntime
YOLOv7 ONNX Runtime C++ inference code

To run code:
```
make
./yolov7-onnxruntime
```


To make the program more flexible, args etc. You can pull request for features.


**Environments**

_Ubuntu 18.04.6 LTS_  
_CUDA 11.4_  
_ONNXRuntime 1.12_  
_OpenCV 4.1_ ( Any version with dnn module )
